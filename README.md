企業綜合點餐平台系統手冊本系統是一套專為企業內部揪團訂餐設計的輕量化解決方案，整合了 前端 React 介面 與 後端 Google Apps Script (GAS)，並以 Google 試算表 作為即時資料庫。🚀 核心功能架構系統主要分為兩個視角：主揪管理視角（主網址）與 同事點餐視角（分享網址）。1. 今日總管 (Daily Dashboard) - 主揪專用餐廳設定：從下拉選單選擇今日揪團的餐廳，系統會自動切換對應菜單。電話管理：輸入店家聯絡電話並儲存至雲端，方便撥打訂購。收款名冊：顯示今日所有同事的點餐明細。繳費勾選：勾選後系統會即時同步至雲端試算表，重新整理也不會遺失紀錄。訂單刪除：可針對錯誤訂單進行單筆刪除。彙整統計：自動加總相同品項的數量，方便直接電話報餐。Excel 匯出：一鍵複製格式化文字，可直接貼上至 Excel 存檔或對帳。分享連結：生成專屬網址，發送給同事即可開始點餐。2. 菜單管理 (Menu Management) - 後台功能價格修改：直接在介面修改餐點單價，儲存後立即生效。品項刪除：移除不再販售的餐點品項。資料同步：所有變動皆與 Google 試算表「菜單資料庫」同步。3. 同事點餐介面 (Order System) - 使用者專用身分選擇：從內建名單選擇自己的姓名。選餐邏輯：左側：負責加減品項數量。右側 (點餐籃)：下單前的最後確認。飲料類：提供冰塊、甜度選項。熱食類：提供「✎ 備註」框（如：不辣、不要香菜）。透明化：下方即時顯示「本日已下單明細」，讓同事知道誰點了什麼。📂 目前整合餐廳菜單小炸物：咖哩系列、手作便當。八方雲集：鍋貼、水餃、乾麵、牛肉麵、湯品。華饌：熱炒類、飯/粥類、麵類。韓亭豆腐煲：韓式拌飯、豆腐煲系列。麻古茶坊：飲中甜品、原味茶系列。🛠️ 技術安裝說明後端 (Google Apps Script)開啟一個 Google 試算表，建立三張工作表：訂單紀錄、菜單資料庫、餐廳電話。點擊「擴充功能」 > 「Apps Script」，貼入最新版本的 Code.gs。點擊「部署」 > 「網頁應用程式」，並取得 SCRIPT_URL。前端 (HTML/React)開啟 index.html。將 const SCRIPT_URL 替換為您從 GAS 取得的網址。直接使用瀏覽器開啟此檔案即可使用。⚠️ 注意事項同步失敗：請確保後端 GAS 部署時，權限設定為「任何人 (Anyone)」，且前端請求使用 Content-Type: text/plain 以避開跨網域限制。標題修復：系統內建自動修復機制，若試算表第一列標題遺失，系統會在下一次寫入時自動補齊。
